# 更新日志

## [1.0.0] - 2025-01-26

### 新增

- 🎉 首次发布 Meta Env Typed
- ✨ 支持 TypeScript 环境变量类型生成
- 🔧 支持 Vite 构建工具集成
- 🔧 支持 Rsbuild 构建工具集成
- 📝 完整的文档和示例

### 特性

- 🛡️ 类型安全的环境变量访问
- 🚀 零配置开箱即用
- 🎯 智能类型推导
- 📦 轻量级，无额外依赖
- 🔄 热更新支持

### 技术栈

- TypeScript 5.0+
- Handlebars 模板引擎
- 支持 ESM 和 CommonJS
